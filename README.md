# Template para aplicativos de Restaurante

Modelo para desenvolvimento de aplicativos de restaurante utilizando as seguintes tecnologias:
- IONIC 3
- NodeJS 6

## Iniciando

Para iniciar o projeto execute os seguintes comandos:
- cd {diretorio_do_app}
- ionic serve
